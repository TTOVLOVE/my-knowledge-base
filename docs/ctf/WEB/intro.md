# 🌐 Web 安全与 CTF 入门指南

在 CTF（夺旗赛）的所有方向中，Web 方向通常是最贴近实际业务场景，也是变化最多端的分支。本模块用于记录在各类靶场和比赛中遇到的 Web 题型、Bypass 技巧以及底层原理分析。

## 🎯 核心技能树

要打好 Web CTF，不仅需要懂得如何“破坏”，更需要深刻理解应用是如何“构建”的：

* **HTTP 协议报文解析：** Header 伪造、Chunked 编码走私、状态码利用。
* **后端语言特性：** * `PHP`：弱类型比较、伪协议、反序列化（POP 链）、无字母数字 RCE。
  * `Java`：内存马注入、Shiro/Fastjson 反序列化、JNDI 注入。
  * `Python`：SSTI 模板注入、Pickle 反序列化、沙箱逃逸。
* **经典漏洞模型：** SQLi, XSS, SSRF, XXE, 文件上传与包含。

## 🧰 必备兵器库

在实战演练中，熟练的工具链能节省大量的时间：

1. **流量抓取与重放：** Burp Suite (必备)
2. **目录与敏感文件扫描：** dirsearch, ffuf
3. **自动化注入验证：** SQLMap
4. **WebShell 管理：** 蚁剑 (AntSword), 冰蝎 (Behinder), 哥斯拉 (Godzilla)

::: tip 💡 学习建议
打 CTF 的核心不在于用工具一把梭，而在于**编写属于自己的 PoC/Exp 脚本**。遇到有意思的赛题，一定要复现并扒开源码分析底层逻辑。
:::
